# Payment Processing Project

## Overview
The Payment Processing Project is a multi-page static website created as a course demonstration. It simulates a payment processing workflow and demonstrates front-end skills using HTML, CSS, and JavaScript, including form validation, responsive design, and user interaction.

Live demo: https://spprofessional.github.io/payment-processing-project/index.html

---

## Pages
| Page | Description |
|------|-------------|
| `index.html` | Home page with payment form |
| `contact.html` | Contact information and form |
| `mission.html` | Project mission / About info |
| `services.html` | Services or project-related info |

--

## Features
- **Responsive design** across desktop and mobile screens  
- **Client-side form validation** for card and contact forms  
- **Simulated payment processing** with JavaScript (no backend)  
- **Error handling and user feedback** for invalid inputs  
- **Multi-page static architecture** with clear navigation

--

## Tech Stack
- **HTML5** – Semantic markup and page structure  
- **CSS3** – Styling and responsive layout  
- **JavaScript** – Form validation and simulated submission

## Usage
1. **Clone the repository**:  
   `git clone https://github.com/SPProfessional/payment-processing-project.git`  
2. **Open `index.html` in your browser**.  
3. Navigate through pages using the top menu.  
4. Fill forms and submit to see client-side validation and simulated confirmation.
